# RSX Weapon HUD

RSX Weapon HUD provides a clean RDR2-inspired interface that displays the player's currently equipped weapon, ammunition count

## Features

* Possiton edit with command /edithud


* Real-time weapon detection
* Dynamic weapon icon support
* Live ammunition count updates
* Ammunition type display
* Minimal screen space usage
* Smooth show/hide transitions
* Supports custom weapon icon packs

## Compatibility
* RedM Standalone
